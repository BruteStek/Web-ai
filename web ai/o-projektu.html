O projektu

Proces tvorby

Celý web byl vytvořen pomocí AI nástroje, který generuje kompletní kód na základě jednoduchých pokynů. Následující kroky popisují proces tvorby:

  1. Výběr tématu a požadavků na web
  2. Generování základní struktury projektu
  3. Tvorba jednotlivých stránek a komponent
  4. Optimalizace pro různé velikosti obrazovek
  5. Přidání interaktivních prvků

Použité technologie

Základní technologie

  • CSS3
  • JavaScript

Pokročilé technologie

  • TailwindCSS
  • Web Components
  • Feather Icons
